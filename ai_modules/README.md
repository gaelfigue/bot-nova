# 🧠 AI Modules — Roadmap

Este directorio albergará los futuros módulos de IA del ecosistema Nova Club.

## Módulos Planificados

### 🎛 `mastering/` — Auto-Mastering con IA
- Análisis espectral automático de tracks
- EQ correctivo basado en perfil de género
- Compresión y limiting inteligente
- Comparación con tracks de referencia

### 📢 `promo_gen/` — Generador de Promos
- Creación de clips promocionales de 15/30s
- Integración con TTS para voiceover
- Templates de visualización para redes sociales
- Generación de waveform animados

---

> **Estado**: En desarrollo. Los módulos se integrarán con el `bot_engine` 
> para ofrecer servicios adicionales a través del bot de Telegram.
